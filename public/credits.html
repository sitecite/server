Node.js libraries used

If you want to use any of these yourself, click on the name and it will open the npm page in a new tab

  • node-canvas: used for generating the images
  • cookie-parser: used for authentication with signed cookies
  • dotenv: used for storing credentials, such as API keys
  • express: used for turning the plain JS files into a webserver
  • express-rate-limit: used to rate limit requests
  • he: used for HTML parsing, making sure that (in theory) no scripts should be able to be executed on the redirect page
  • mysql2: used for communicating with the MySQL database
  • node-fetch: used to ensure compatibility with older versions of node.js by adding fetch()
  • yaml: used for reading the config file
